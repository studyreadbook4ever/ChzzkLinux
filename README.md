# ChzzkLinux
리눅스에서 치지직 방송화면만 낮은 메모리활용으로 봐요 (ArchLinux Chromium 기준 크롬창에 치지직 하나 띄우면 메모리사용 900MB~1200MB 정도, 이 ChzzkLinux는 280~400MB정도 )

사용법- 실행 파일들 전부 다운받고, channels.toml 파일에 "부르고 싶은 별명" = "치지직 링크의 채널코드" 형태로 (예시: "허츄" ="c0d9723cbb75dc223c6aa8a9d4f56002" 식으로 ) 넣은 뒤에 

## ./launcher.sh <채널 별명> <설정하고 싶은 화질(1080/720/480/360/144, 디폴트==1080)> 식으로 실행하면 됩니다.

개발 런타임: 구상 2시간 손코딩 5시간(main.rs) 후 AI한테 호환성-mpv 최적화 관련 옵션도움받기 1시간 + .sh 바이트코딩 3시간(ing... Ubuntu-Fedora-Arch를 동시에 지원시키고 싶은데 ㅠㅠㅠ 이게 힘들무ㅠㅠㅠ)

리눅스의 기본적으로 내장된 webKitGTK를 적극적으로 활용해 Chromium의 오버헤드를 줄였습니다
